Destructuring

destructuring is a process of unpacking the values from the array or object into the distinct variables.

2 types of destructuring

1:Array Destructuring : it is a process of unpacking the values from array into the distinct variables.
Allows the programmer to access the data meaningfully from the array, without using the index.
SYNTAX : let [val1,val2,val3...] = [value1,value2,value3 ....value n]

2:Object Destructuring
object destructering is a process of unpacking the values from the object without using the object name.
while destructuring the variables must be same as keys.
SYNTAX: let {id,name,place} = { id:1,name:"ram",place:"ayodhya"}


Timing Functions

Timing Functions are used to execute a piece of code in the required interval of the time or to execute after some delay.
in javascript there are 4 timing functions
1: setInterval() 2:clearInterval() 3:setTimeout() 4:clearTimeout()
setInterval(), clearInterval()
setInterval() is used to execute a piece of js code continuously for a given interval of time.
it accept 2 arguments that is callback, time. setInterval(callback,time)
